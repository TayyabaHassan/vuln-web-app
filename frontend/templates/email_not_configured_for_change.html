Security Vulnerability Lab

Email delivery is not configured

Changing your email address requires the server to send a confirmation link to the new address. This server does not have email delivery set up, so the email-change feature is unavailable right now.

To enable this feature, set the SENDGRID_API_KEY and SENDGRID_FROM environment variables (or the matching entries in a git-ignored .env file) and restart the application. See core/config.py for the full list of supported settings.
